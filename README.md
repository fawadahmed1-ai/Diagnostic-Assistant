MediGraph Diagnostic Assistant 

MediGraph is an AI-powered diagnostic assistant designed specifically for chest X-rays and radiology reports. It lets doctors and radiologists instantly search their image and report database using natural language descriptions (e.g. 'cardiomegaly', 'bilateral infiltrates', 'normal chest') or by uploading a new X-ray — and get the most visually and textually similar past cases in seconds.

Key strengths:

Multimodal CLIP-based search (text + image similarity)
Hybrid mode (combine text description + uploaded image for better results)
Full report text displayed + AI-generated neutral relevance summary (via Groq LLM)
Clean, responsive UI with tabs, zoomable images, clear button
Already deployed on Streamlit Cloud and indexed with 200+ real cases (scalable to thousands)
It's not a diagnostic tool — it's a fast, intelligent second-opinion and reference assistant that helps clinicians quickly recall similar cases, compare findings, and reduce lookup time from minutes to seconds."
